About this project

Mosaic is a small full-stack web application built for CSE 4500. Users create albums, upload images, view them in a full-screen lightbox, and delete what they no longer want. The backend persists every album and image to a SQLite database, and stores uploaded files on a disk volume that survives redeploys.

Tech stack

  • Node.js 22
  • Express 4
  • SQLite (better-sqlite3)
  • Multer (file uploads)
  • Vanilla JS / CSS
  • Render (deploy)
